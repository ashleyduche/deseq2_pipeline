# DGE Analysis Using DESeq2

Generate gene-expression signatures with DESeq2, before and after ComBat-seq batch
correction, using `deseq2_pipeline/scripts/deseq2_pipeline.R`.

## Repository structure

```
deseq2_pipeline/
├── scripts/
│   └── deseq2_pipeline.R      Two-stage DESeq2 pipeline (raw counts, then ComBat-seq-corrected)
├── data/
│   ├── 2023.GFP18_AKT_FeatureCounts.csv   Raw gene-level counts (AKT vs. GFP control)
│   └── 2023.GFP18_AKT_annotations.csv     Sample annotations (condition, batch)
└── results/                    Written by the script: result tables, MA/PCA plots, sessionInfo.txt
```

## Requirements

- R (≥ 4.0)
- Bioconductor: `DESeq2`, `sva`
- CRAN: `ggplot2`

`deseq2_pipeline.R` installs any of these that are missing on first run (Bioconductor
packages via `BiocManager`, not plain `install.packages()`).

## Quickstart

```bash
cd /path/to/parent/of/deseq2_pipeline
Rscript deseq2_pipeline/scripts/deseq2_pipeline.R
```

To run on a different dataset, edit the `CONFIG` block at the top of the script
(`counts_path`, `coldata_path`, `condition_col`, `reference_level`, `batch_col`) —
nothing else needs to change.

## Example dataset

1. **Counts**: `2023.GFP18_AKT_FeatureCounts.csv` — gene-level feature counts.
2. **Annotations**: `2023.GFP18_AKT_annotations.csv` — sample metadata (condition, batch)
   used to build DESeq2's design and to run ComBat-seq.

### Background

This example dataset profiles a GFRN pathway component, **AKT**, against its matched
Green Fluorescent Protein (GFP) control samples. The signature was generated by
overexpressing GFRN pathway-specific genes in human mammary epithelial cells (HMECs), as
described in ["Activity of distinct growth factor receptor network components in breast
tumors uncovers two biologically relevant subtypes"](https://www.ncbi.nlm.nih.gov/pmc/articles/PMC5406893/).

Raw sequencing data is from NCBI GEO accession
[GSE83083](https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi?acc=GSE83083):

- FASTQ files retrieved via SRA Run Selector for GSE83083.
- Reads trimmed with [Trimmomatic](http://www.usadellab.org/cms/?page=trimmomatic) (bioconda).
- Reads aligned and quantified with Bioconductor's
  [Rsubread](https://bioconductor.org/packages/devel/bioc/vignettes/Rsubread/inst/doc/Rsubread.pdf)
  package, against the **GRCh38** reference genome.

## Output

Running the script writes to `deseq2_pipeline/results/`:

- `raw_results.csv`, `combat_corrected_results.csv` — DESeq2 results tables, before/after
  batch correction.
- `raw_MA_plot.png`, `combat_corrected_MA_plot.png` — MA plots.
- `raw_PCA.png`, `combat_corrected_PCA.png` — sample PCA, colored by condition.
- `combat_seq_corrected_counts.csv` — the ComBat-seq-corrected count matrix.
- `sessionInfo.txt` — exact package versions used, for reproducibility.
